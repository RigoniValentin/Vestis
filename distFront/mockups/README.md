# Mockups de Prendas

Esta carpeta debe contener 4 imágenes PNG con fondo transparente para la vista previa del constructor de remeras personalizadas.

## Imágenes requeridas:

### Remeras (con mangas cortas):
1. **remera-blanco.png** - Remera blanca, vista frontal, fondo transparente
2. **remera-negro.png** - Remera negra, vista frontal, fondo transparente

### Musculosas (sin mangas):
3. **musculosa-blanco.png** - Musculosa blanca, vista frontal, fondo transparente
4. **musculosa-negro.png** - Musculosa negra, vista frontal, fondo transparente

## Especificaciones técnicas:

- **Formato:** PNG con transparencia (canal alpha)
- **Tamaño recomendado:** 1000x1200px (o proporción similar)
- **Vista:** Frontal centrada
- **Fondo:** Completamente transparente
- **Calidad:** Alta resolución para zoom/responsive
- **Ángulo:** Prenda plana o en maniquí invisible
- **Área de diseño:** El pecho debe estar claramente visible para superponer diseños

## Recursos para conseguir mockups:

### Gratuitos:
- **Placeit.net** - Mockups gratuitos limitados
- **Mockup World** - Gran variedad gratuita
- **Freepik** - Mockups con atribución
- **Vecteezy** - Vectores y mockups gratis

### De pago (premium):
- **Placeit.net** - Suscripción mensual
- **Smartmockups.com** - Generador online
- **Creative Market** - Mockups individuales
- **Envato Elements** - Suscripción con miles de mockups

### Herramientas para remover fondo:
- **remove.bg** - Automático, muy preciso
- **Photopea** - Editor gratuito online (como Photoshop)
- **GIMP** - Software gratuito para edición

## Posicionamiento del diseño:

El diseño se superpondrá automáticamente en:
- **Posición vertical:** 36% desde arriba (zona del pecho)
- **Posición horizontal:** Centro exacto
- **Tamaño:** 28% del ancho de la prenda
- **Efecto visual:** 
  - Remera blanca: multiply blend mode (diseño se integra)
  - Remera negra: screen blend mode + brightness (diseño resalta)

## Ejemplo de nomenclatura correcta:

```
/public/mockups/
├── remera-blanco.png
├── remera-negro.png
├── musculosa-blanco.png
└── musculosa-negro.png
```

## Fallback temporal:

Mientras consigues las imágenes, el sistema usa placeholders de Unsplash automáticamente.

## Tips para mejores resultados:

1. ✅ Usa imágenes de prendas lisas sin estampados previos
2. ✅ Asegúrate de que el pecho esté centrado y visible
3. ✅ Mantén proporción consistente entre todas las imágenes
4. ✅ Verifica que la transparencia sea limpia (sin halos blancos)
5. ✅ Prueba el resultado en el preview antes de finalizar
